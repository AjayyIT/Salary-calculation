📌 Overview:
The Employee Salary Calculator is a Python-based application designed to calculate an employee's net salary after considering allowances, deductions, and taxes.

This project demonstrates:
• Python functions
• Exception handling
• Input validation
• Arithmetic operations
• User interaction through console input

🚀 Features:
• Calculates gross salary
• Calculates tax deductions
• Computes final net salary
• Handles invalid input safely
• Supports positive number validation
• Beginner-friendly implementation

🛠 Technologies Used:
• Python
• Exception Handling
• Functions
• Console-based Interaction

⚙️ How It Works
1. User enters:
  • Basic Pay
  • Allowances
  • Deductions
  • Tax Percentage
2. The program:
  • Calculates gross salary
  • Applies deductions and taxes
  • Displays the final net salary

🧠 Concepts Used:
• Functions
• Loops
• Exception Handling
• Conditional Statements
• Mathematical Computation
